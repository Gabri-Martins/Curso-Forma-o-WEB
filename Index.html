O que é o HTML

HTML: Hyper text Markup Language, é o bloco de construção, atraves dele é possivel estruturar o conteudo para um site web.

Durante o curso aprendi a inserir tags e formatar paragrafos inserindo negrito, itálico e sublinhado. Alem de inserir ancoras bem como links de acesso a site e inserção de imagem

O que são Tags?

Tags: Tags são elementos utilizados em HTML para formatação do texto ou no visual, no curso tive a oportunidade de conhecer as tags basicas. Conheci tambem o site da W3C para utilizar e buscar maiores informações sobre o HTML e toda sua extensibilidade.

O que são Clients?

Clients: Clients em HTML é o usuario final, quem de fato acessa os sites e navega na internet, nessa parte do curso conhecemos melhor os navegadores e visão do lado Client.

O que são Servers?

Servers: Servers é o lado que armazena e disponibiliza o conteudo de um site para os clients acessarem são armazenados em servidores com maior poder computacional podem ainda estar ligados a bancos de dados.

O que são Lists

Lists: List é uma tag usada para inserir listas em uma estrutura HTML.Pode-se usar os comanado OL ou UL para se construir listas.
